LWood Audio Catalog Creator

Documentation Overview

Welcome to the Audio Catalog Generator. This simple to use generator will create an HTML page that will list, describe, and play your audio files. You can then simply copy and paste the public folder on any website as HTML files. Advanced users may use github to host your audio catalog for FREE.

HThe HTML generator is SEO friendly,. That means google and other search engines will be able to find them, and you will get noticed. Those that produce musical content can provide a lower quality or excerpt of the actual music. Plus you can add links for where they can purchase it or get use rights.

Here your users can play the audio files. Download the audio files. Bookmark, share and so on. This genrator requires no HTML or css knowledge.

What you will need

  • text editor - Any will do.
  • NodeJS - No programming needed just needs to be installed to generate the files. Select version for your system (Windows, Linux, Mac)
  • Media Audio files such as mp3's, Wav, and so on.
  • Cloud Drive, WebSite, or GitHub if you want to share your Audio Catalog with the world.

Optional

Html/Css editor. This is for those that are comfortable making such modifications. It is not needed if you are okay with the Audio Catalog layout & colors I provided.

GitHub account. This will allow you to dowload the entire project and host the html free for yourself

SSH or GPC configured and linked to your github account. This will will allow automatic updates and uploads. This is very Advanced you may need help possibly. You still other options.

Quick Steps

  1. npm run start
  2. Go to http://localhost:5000/templates
  3. Add media files and/or modify descriptor files
  4. Repeat step 2 Templates until done.
  5. Go to http://localhost:5000//api/import
  6. Go to http://localhost:5000/make-html
  7. Go to http://localhost:5000/make-html?save=true
  8. Shut down this server. ctrl + c
  9. Public/index.html is the catalog
